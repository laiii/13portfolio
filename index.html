Interactive Installation

DataSync

Sound and graphic have ofcourse been the very common forms of data visualization. But what can be more that them?

Data Sync is an interactive project trying to experiment data visualization in new forms. It takes datas including sounds, light, temperture and humidity from the near environment. Beside translate data into sounds and 3D graphic, Data Sync also use it to grab and filter infomation in a big scale. You can see from the demo video that various results of stock index and news headlines are selected and display, due to different combinations of data input.

This project is inspired by Chilean Socialism project Cybersyn. This installation has been exhibited in V&A Mesuem, London in July, 2013.
